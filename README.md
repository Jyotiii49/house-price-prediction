# 🏠 House Price Prediction

## 📌 Project Overview

This project predicts house prices using Machine Learning (Linear Regression) based on the Boston Housing Dataset.

## 📊 Features Used

* CRIM (Crime rate)
* RM (Number of rooms)
* TAX (Property tax)
* LSTAT (Lower status population)
* and more...

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## 🧠 Model Used

Linear Regression

## 📈 Evaluation Metrics

* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

## 📂 Project Files

* house_price_prediction.ipynb

## 🎯 Conclusion

The model successfully predicts house prices based on given features.
